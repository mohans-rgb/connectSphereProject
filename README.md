# ConnectSphere

A LinkedIn-inspired social networking backend built with Java Spring Boot microservices architecture.

## Services
- **API Gateway** – Routes requests and handles JWT authentication
- **User Service** – User registration, login, and profile management
- **Posts Service** – Create and manage posts with media uploads
- **Connections Service** – Follow/connection graph powered by Neo4j
- **Notification Service** – Real-time notifications via Kafka
- **Uploader Service** – Media uploads via Cloudinary / Google Cloud Storage
- **Discovery Server** – Eureka service registry

## Tech Stack
Java · Spring Boot · Spring Cloud · PostgreSQL · Neo4j · Apache Kafka · Docker · Kubernetes
